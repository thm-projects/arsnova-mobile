How to define templates for the ARSnova moderation question
-----------------------------------------------------------

The template.json file is the central maintenance point of the ARSnova template system.
The offered templates are described in the form of a JSON-Array as seen below. A localization-system allows to name and describe the
templates in different languages. 

Currently the example shows all possible keys. The key names match the attribute names of the class generated by the template system. It does only make sense to use keys which corresponding attributes are saved in the database, though otherwise they would be lost when a question with the templates is created.
The template system (or the grid system in general) currently supports JPEG and PNG encoded images, whereas JPEG encoding is preferred.


Example:
--------

[
    {
		"name": {
			"de": "Titel der Vorlage",
			"en": "Template title"
		},
    	"description": {
    		"de": "Hier können Sie eine Beschreibung für das Template eingeben. Diese ist insbesondere wichtig, wenn das eigentliche - nicht sichtbare - Raster sich nicht aus dem Bild erschließt.",	
        	"en": "Here you can describe further details of the template. This may be especially important in case the (invisible) grid might not be recognized at first sight."
        },
        "imageFile": "myTemplate.jpg",
        "gridSize": 16,
        "scaleFactor": 1.2,
        "scale": 1,
        "zoomLvl": 2,
        "offsetX": 4,
        "offsetY": 3,
        "gridOffsetX": 1,
        "gridOffsetY": 1,
        "gridZoomLvl": 3,
        "gridSizeX": 8,
        "gridSizeY": 4,
        "gridScale": 2,
        "imgRotation": 1,
        "cvBackgroundColor": "#C0FFEE",
        "cvIsColored": true
    },
    ...
]
